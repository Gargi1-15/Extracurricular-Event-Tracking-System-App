# Extracurricular-Event-Tracking-System-App
The Extracurricular Event Tracking System is a web-based application designed to manage and monitor student participation in various non-academic activities such as sports, cultural events, and technical competitions. The system allows administrators to create and manage events, while students can view available events and register for them online. It maintains a centralized database to track participation records, making it easier to analyze student involvement and engagement. This application simplifies event management, reduces manual work, and provides an efficient way to organize and track extracurricular activities within an institution.
